How the math works. A European wheel has 37 pockets (0 to 36). A single number pays 35 to 1 but wins only 1 time in 37, so its true value is 36/37 = 97.30%. Every bet on the table is built to return the same 97.30%, which means the house keeps 2.70% on average. That fixed edge, identical across all bets, is the whole design of the game.
